# (Bid) Create Bid


/sentadv-ads-bid/api/v1/bid

This endpoint allows you to make a bid.

# Request

  • Headers
    • Authorization
      • Required: true
      • Type: string
      • Description: Authorization token.
  • Query String
    • app_id
      • Required: true
      • Type: string
      • Description: User's AppID
    • playtimeid
      • Required: false
      • Type: string
      • Description: Video playtime type
  • Body
    • campaign_criteria_id
      • Required: true
      • Type: string
      • Description: Campaign criteria ID of campaign to bid
    • ads_balance_id
      • Required: true
      • Type: string
      • Description: Ads balance ID of Campaign criteria to bid
    • amount
      • Required: true
      • Type: integer
      • Description: Bid's amount

# Response

  • Type: JSON
  • 200: OK
    {
  "data": {
    "Status": 200,
    "Message": "Success"
  }
}
  • 400: Insufficient balance
    {
  "error_message": "Your balance is not enough to bid",
  "status_code": 400
}
  • 404: Campaign Criteria ID or Ads Balance ID Not Found
    {
  "errors": [
    {
      "status": "404",
      "code": "404",
      "title": "Not Found",
      "detail": "Ad not found",
      "source": {
        "parameter": "/bid"
      }
    }
  ]
}

# (Hedwig) Send Email

      • Type: string
      • Description: Email subject
    • recipient
      • Required: false
      • Type: string
      • Description: Email recipient
    • cc
      • Required: false
      • Type: string
      • Description: Email CC recipient
    • bcc
      • Required: false
      • Type: string
      • Description: Email BCC recipient
    • body
      • Required: false
      • Type: string
      • Description: Email body
    • user_id
      • Required: false
      • Type: string
      • Description: Sender's UserID
    • advertiser_id
      • Required: false
      • Type: string
      • Description: Sender's AdvertiserID

      • Required: false

      • Description: Sender's AppID
