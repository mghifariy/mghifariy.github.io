# (Comments) Create Comment by Type and Reference ID

@post http://stag.supersoccer.tv/api/v2/discussions /comments/{type}/{refID}

This endpoint allows you to create comment by type and reference ID.

Params Description
type video / playlist / reply
refID Reference ID

@title Request Params

@param app_id string required Application ID

@title Request Headers

@param Content-Type string required Entity header is used to indicate the media type of the resource

@param Authorization string required Authorization token

@title Request Body

@param content string required Content of comment

  • Type: JSON
    {
    "content": "mantap"
}

@title Response

  • Type: JSON

  • 200: OK

    {
    "id": 18,
    "userName": "Mohammad Ghifari Yusuf",
    "content": "mantap",
    "updatedAt": "2018-12-27T16:26:21.427867+07:00"

    {

}

  • 400: Bad Request

    {
    "error": "Comment can't be empty"
}
