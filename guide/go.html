# Formatting

All of our go codebases must be in standard Go formatting and follow everything mentioned in Effective Go.

To format go code, use gofmt command. https://blog.golang.org/go-fmt-your-code

Or, use IDE's extension. https://code.visualstudio.com/docs/languages/go#_formatting

# Conventions

  • Use camelCase instead of snake_case
  • Tabs over spaces
