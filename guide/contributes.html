# Prerequisite

  • Node.js >= 8.

# Clone project

Clone project's repository and go to working directory

git clone git@git.sstv.io:docs/guidelines.git && cd guidelines

# Install node modules

# Install node modules using NPM

npm i

# Install node modules using Yarn

yarn

# Run development server

# Run using NPM

npm run dev

# Run using Yarn

yarn dev

# Commit project

  1. Commit your project and push to develop branch
  2. Merge request to master branch in project's repository
